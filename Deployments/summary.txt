PM2 -> Gestor de despligues
SERVE -> Crear servidor

COMANDOS PM2
pm2 start [file.js] -n [name] -> Crear un nuevo proceso
pm2 stop [name|id] -> Detener proceso
pm2 delete [name|id] -> Eliminar proceso
pm2 delete all -> Eliminar todos los procesos
pm2 log [name|id] -> Ver logs del proceso
pm2 log -> Ver los logs de todos los procesos
pm2 env [name|id] -> Ver las variables de entorno

COMANDOS SERVE
serve -s [build_folder_path] [port] -> Iniciar servidor en puerto especificado (si no se especifica, por defecto es 3000)


PM2 + SERVE
pm2 serve [build_folder_path] [port] --spa --name [name_process]
