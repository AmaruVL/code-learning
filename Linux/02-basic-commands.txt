pwd -> Print working directory
ls -> List
ls -a -> Listar todos los directorios (incluido los ocultos)
ls -l -> Listar directorios con sus permisos
ls -al -> Mostrar directorios (tambien ocultos) con sus permisos
ls -al file -> Mostrar -al de file

// Moverse de directorios
/ -> Raiz
~ > /home/user
cd ~ -> Ir a la directorio de usuario
cd / -> Ir a la raiz del sistema
cd Directorio/ -> Ruta relativa
cd /home/user/directorio -> Ruta absoluta (comienza con /)
